# My Study Planner 

A streamlined application designed to help students and lifelong learners organize their academic life, track progress, and stay on top of deadlines.

## Features

- **Task Management:** Create, edit, and delete study tasks.
- **Schedule Overview:** Visualize your weekly or daily study routine.
- **Deadline Tracking:** Set due dates and receive reminders for upcoming exams.
- **Google:** Can save to Google Calender.

## Deployed Link

https://jyo-2028.github.io/My_Study_Planner/

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript 
- **Database:** LocalStorage
- **Deployment:** GitHub Pages

# My Study Planner 

A streamlined application designed to help students and lifelong learners organize their academic life, track progress, and stay on top of deadlines.

## Features

- **Task Management:** Create, edit, and delete study tasks or assignments.
- **Schedule Overview:** Visualize your weekly or daily study routine.
- **Deadline Tracking:** Set due dates and receive reminders for upcoming exams or project submissions.


## Tech Stack

- **Frontend:** HTML, CSS, JavaScript 
- **Database:** LocalStorage
- **Deployment:** GitHub Pages
